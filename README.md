## 專案名稱：美術館資料檢索網

此專案為建立一個可供瀏覽及搜尋的資料庫展示介面，輔以適當的美化及簡潔的介面供人員操作

---
### 專案簡介
- 專案目標

建立一個可供後台人員使用的資料庫介面，達成抽象化 (Abstraction) 之目的

- 專案流程
1. 建立 MySQL 資料庫，各資料表須符合四大正規化規範
2. 導入藝術家、顧客及作品資料並規劃相應的 relation
3. 以簡單的 Html 模板建立簡潔的閱覽頁面
4. 以 PHP 做為後端程式碼，負責調用資料庫及呈現資料，完成此專案
